El elemento HTML 'Mark'

Este elemento no tiene atributo que no sean los atributos globales, comunes a todos los elementos.

No se debe confundir el elemento mark con el elemento strong. El elemento strong se utiliza para denotar intervalos de texto de especial importancia, mientras que el elemento mark se utiliza para denotar intervalos de texto de especial relevancia.
